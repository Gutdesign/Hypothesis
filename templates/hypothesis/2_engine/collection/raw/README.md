# Raw collection outputs go here.

This directory holds unprocessed outputs from:
- Parallel.ai query responses (as `.md` or `.json`)
- Apify scraping run outputs
- web_fetch retrievals of specific articles or pages

The synthesized files at the parent level (`competitors.md`, `market.md`, `signals.md`) reference these but are independently readable.

Files in this directory may be large. Binary files are git-ignored via the root `.gitignore`.
