# 02 — Assumptions

> **Zone:** 1_frame (founder-led, with Claude as challenger)
> **When to fill:** after the hypothesis is sharpened.
> **Purpose:** make every load-bearing assumption explicit so each can be tested separately.

---

## How to use this file

For each assumption: state it, then state what would have to be true for it to hold, then state what evidence would refute it.

Ask Claude: "What assumptions is my hypothesis quietly making?" Then add anything missed.

The goal is a list where, if you crossed out any single assumption, the hypothesis would fall apart. Anything that can be removed without consequence isn't actually load-bearing — drop it.
