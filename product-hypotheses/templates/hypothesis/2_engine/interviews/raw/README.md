# Raw interview materials go here.

This directory holds:
- Audio recordings of interviews
- Unedited transcripts
- Notes taken during the conversation

Audio files are git-ignored (see root `.gitignore`). Transcripts as `.md` or `.txt` are committed.

Naming convention: `interview_NN_[brief_identifier].[ext]`

The processed synthesis lives one level up in `synthesis.md`, which references files in this directory but is independently readable.
